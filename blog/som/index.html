Business Process Modeling with the Semantic Object Model

2017, Jan 07

Richard Herz, Andreas Steffan and I authored a book chapter on the introduction of the Semantic Object Model (SOM) using a case study based on the zooplus AG.

The german language publication is available at Springer Link. Further information on the Semantic Object Model by Ferstl and Sinz can be found in this english language publication.

Härer F., Steffan A., Herz R.A. (2016) Fallstudienbasierte Einführung von SOM. In: Benker T., Jürck C., Wolf M. (eds) Geschäftsprozessorientierte Systementwicklung. Springer Vieweg, Wiesbaden

Abstract: Das Semantische Objektmodell ist eine Methodik zur Modellierung betrieblicher Systeme, deren Anwendung nach einer allgemeinen Einführung anhand einer Fallstudie demonstriert wird. Diese umfasst die Modellierung von Unternehmensplan und Geschäftsprozess sowie die Spezifikation von Anwendungssystemen.
